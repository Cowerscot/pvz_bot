# Рефакторинг PVZ Bot

## Структура проекта

```
/opt/pvz-bot/
├── main.py              # ТОЧКА ВХОДА - запускать вместо bot_engine.py
├── config.py            # Публичные настройки (импортирует secrets)
├── secrets.py           # Чувствительные данные (НЕ КОММИТИТЬ!)
├── database.py          # База данных: пользователи, cookies, сессии
├── browser_manager.py   # Управление браузером Selenium
├── vk_bot.py            # VK бот: обработчики команд, клавиатуры
├── yandex_core.py       # Логика Яндекс.ПВЗ (исправлена!)
├── ozon_core.py         # Логика Ozon (требует доработки)
├── avito_core.py        # Логика Avito (требует доработки)
└── .gitignore           # secrets.py добавлен в игнор
```

## Что исправлено

### 1. Разделение чувствительных данных
- **secrets.py** — токены, пароли, ключи (НЕ КОММИТИТЬ!)
- **config.py** — публичные настройки, импортирует secrets

### 2. Надёжное сохранение сессий
- **database.py** — проверка срока действия cookies (Session_id expiry)
- Сессия не теряется при перезапуске если cookies действительны
- Отдельные колонки для каждого маркетплейса: yandex_cookies, ozon_cookies, avito_cookies

### 3. Исправлена проблема "✅ Вы уже авторизованы"
- **vk_bot.py** — проверка cookies ПЕРЕД запуском авторизации
- Кнопка "🔄 Переподключить Яндекс" теперь работает — сначала очищает cookies

### 4. Кнопка очистки данных
- "🗑️ Очистить данные Яндекс" — сброс cookies Яндекса
- "🗑️ Очистить все данные" — полный сброс пользователя

### 5. Исправлена логика скачивания Яндекс
- **yandex_core.py** — правильный путь загрузки `/home/seluser/Downloads`
- Увеличен таймаут ожидания файла: 30 → 60 секунд
- Проверка обеих папок (локальной и контейнера Selenium)
- Логирование содержимого папок при ошибке

### 6. Модульная архитектура
- **main.py** — точка входа (заменять bot_engine.py)
- Разделение ответственности между модулями

## Запуск

```bash
# Вместо:
/opt/pvz-bot/venv/bin/python /opt/pvz-bot/bot_engine.py

# Запускать:
/opt/pvz-bot/venv/bin/python /opt/pvz-bot/main.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы |
| `✅ Принимаю политику` | Подтверждение политики |
| `🟡 Яндекс ПВЗ` | Авторизация в Яндекс |
| `🔵 Ozon ПВЗ` | Авторизация в Ozon (в разработке) |
| `🟢 Avito ПВЗ` | Авторизация в Avito (в разработке) |
| `📊 Статистика Яндекс` | Получить статистику |
| `🔄 Переподключить Яндекс` | Сброс сессии и новая авторизация |
| `🗑️ Очистить данные Яндекс` | Удалить cookies Яндекс |
| `🗑️ Очистить все данные` | Полный сброс пользователя |
